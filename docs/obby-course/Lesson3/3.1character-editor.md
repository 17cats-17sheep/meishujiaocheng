# 角色编辑器

## 角色编辑工具是什么

角色编辑工具为开发者提供可视化的角色形象编辑能力，通过调整工具中体型、发型、服饰、妆容等选项可以对游戏中的角色形象进行个性化定制。

### 头盔插槽部分想要游戏中显示

虽然编辑器中我们只做了头盔，但是实际游戏运行起来我们是看不见头盔的，还需要一部分代码支持

```ts
let humanoidV2 = this.Character.setAppearance(Gameplay.HumanoidV2)
// 鸭子骑士的工程内容ID
humanoidV2.setAppearanceData(["52C2B783418E2CA9E1F635B79C162348"], () => {
	// 不调用这一行，其他玩家将看不到自己换装
	humanoidV2.appearanceSync()
	// 设置鸭子骑士的插槽数据
	humanoidV2.setSlot("52C2B783418E2CA9E1F635B79C162348", true)
	// 设置头盔
	Gameplay.GameObject.asyncSpawn({guid: "44286"}).then(go=>{
		humanoidV2.attach(go, Gameplay.SlotType.Rings)
	})
	// 设置鸭子
	Gameplay.GameObject.asyncSpawn({guid: "92715"}).then(go=>{
		humanoidV2.attach(go, Gameplay.SlotType.Rings)
	})
	// 设置另一只鸭子
	Gameplay.GameObject.asyncSpawn({guid: "89151"}).then(go=>{
		humanoidV2.attach(go, Gameplay.SlotType.Rings)
	})
})
```

代码中的**52C2B783418E2CA9E1F635B79C162348**为新保存的角色的**工程内容ID**
