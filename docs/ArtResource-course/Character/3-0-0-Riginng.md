## 软件推荐

- 3D Max（**2019**以上版本）
- 可通用：Maya、Blender（但是需要使用提供的 FBX 文件）

## 准备工作：

- 检查模型坐标是否归零，未归零及时进行修改
- 检查模型朝向，软件正视图和模型面部正方向一致即可
- 检查单位设置，统一设置为厘米![img](https://arkimg.ark.online/1690962720807-1.png)
