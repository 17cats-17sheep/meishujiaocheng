## 不同类型的区别：

|          |          基础人形形象          |          高级人形形象          |          多足形象（研发中）          |
| :------: | :----------------------------: | :----------------------------: | :----------------------------------: |
|   换装   |             不支持             |            支持换装            |        仅限同种骨骼类型内支持        |
|   材质   |              一个              |              多个              |                 一个                 |
|   贴图   | 一套（不同风格类型，数量不同） | 多套（不同风格类型，数量不同） |    一套（不同风格类型，数量不同）    |
|   骨骼   |   基础骨骼——MW_Skeleton骨骼    |   基础骨骼——MW_Skeleton骨骼    | 独立骨骼（按骨骼类型分类，分类内部） |
|   动画   |        MW_Skeleton动画         |        MW_Skeleton动画         |               独立动画               |
| 动态骨骼 |             不支持             |              支持              |                不支持                |
| 捏脸捏体 |             不支持             |              支持              |                不支持                |
|  布娃娃  |              支持              |              支持              |                 支持                 |
|   挂件   |             不支持             |              支持              |            当前版本不支持            |
