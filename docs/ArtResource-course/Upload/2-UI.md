- 点击同步到工程：先用后审，会在发布游戏的时候再审核资源并自动上传。（也可同步到工程之后右键资源点上传）
- 点击上传资源：直接上传并使用。可以在传作者中心看到我的资源状态。

资源状态：传作者中心-资源-我的资源
