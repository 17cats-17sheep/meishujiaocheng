### 2.蒙皮绑定：

- **必须**使用提供的**骨架**进行模型适配
- 对模型添加**Skin**，添加相关联的骨骼，骨骼**Bip_01**和根节点**Root**不参与蒙皮
- 蒙皮时**骨骼**影响**限制数为3**
- 模型使用等比例缩放基础骨架，关节点保持一致比例，绑定资源后期可以创建独立动画
- 模型使用非等比例缩放基础骨架，关节点可以不一致，绑定资源后期只能使用通用动画
- 绑定完成后，可通过旋转骨骼来测试蒙皮效果，根据效果进行调整优化
