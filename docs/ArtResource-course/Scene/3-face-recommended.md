# 面数推荐

**单个静态模型上限36000三角面，原则性不限制面数，使用推荐面数可以最大化利用编辑器性能。**

## 高性能面数推荐

|              | 推荐面数（三角面） |
| ------------ | ------------------ |
| 建筑模型     | ＜10000            |
| 景观模型     | ＜3000             |
| 树           | ＜5000             |
| 武器模型     | ＜3000             |
| 载具模型     | ＜15000            |
| 家具         | ＜6000             |
| 小型道具     | ＜1000             |
| 中型道具     | ＜4000             |
| 大型道具     | ＜8000             |
| 远景（面片） | 20~3000            |
